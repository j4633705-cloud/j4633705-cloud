## 🚀 Projetos

<details>
<summary><strong>🤖 PrismCode — Agente de IA para terminal</strong></summary>
<br>

Agente de coding para terminal com suporte a 18+ provedores de LLM. Sistema de skills em JSON, modos PLAN/BUILD, BYOK + gateways, créditos via Polar.sh e binários pré-compilados para todas as plataformas.

**Stack:** TypeScript · Bun · Hono · Vercel AI SDK · Prisma · Clerk · Polar.sh

🔗 [prismcode.dev](https://prismcode.dev) · [npm](https://www.npmjs.com/package/prismcode)

</details>

<details>
<summary><strong>✍️ SignFlow — Assinatura de documentos para freelancers brasileiros</strong></summary>
<br>

Plataforma de assinatura de documentos pensada pro contexto brasileiro. Webhooks com HMAC/SSRF protection, automações de eventos, OAuth Slack, domínios customizados e Stripe. Filosofia: *abriu, assinou, acabou.*

**Stack:** TypeScript · Next.js · Prisma · Stripe · PostgreSQL

</details>

<details>
<summary><strong>🦾 Desktop Robot — Robô de mesa com ESP32-S3</strong></summary>
<br>

Robô de mesa com olhos animados no OLED, reações ao toque capacitivo e movimentos senoidais nos 4 servos SG90. Firmware adaptado do STM32 original para ESP32-S3 com animações customizadas.

**Stack:** C++ · ESP32-S3 · SSD1306 · SG90 · 3D Print

</details>

<details>
<summary><strong>🔐 Auth Boilerplate — Sistema de autenticação completo</strong></summary>
<br>

Sistema de autenticação com refresh token rotation, prevenção de timing attacks, tokens em memória e 26 testes de integração. Migrado integralmente para TypeScript.

**Stack:** TypeScript · Node.js · SQLite · JWT

</details>
